وصف المشروع
==========================
بوت تيليجرام يقدم خدمة البحث عن الدورات، رفع الدورات، وإدارة نظام نقاط يعتمد على المكافآت والدعوات. يعتمد على Python وPostgreSQL وSerpAPI، ويعمل بواجهة عربية كاملة باستخدام ReplyKeyboard.

المميزات
==========================
واجهة عربية كاملة
بحث عن الدورات عبر SerpAPI
خصم نقاط قبل تنفيذ البحث
نظام نقاط (تسجيل – دعوات – يومي – مشاركة)
شحن النقاط عبر TON
رفع الدورات وإضافتها للمفضلة
عرض حساب المستخدم
لوحة إدارة للمشرف
تنبيه المشرف عند حدوث أخطاء أو توقف الخدمات
قاعدة بيانات PostgreSQL جاهزة للتوسع

خطوات التشغيل

تثبيت الحزم
pip install -r requirements.txt

تشغيل البوت
python main.py

رفع المشروع على Render

رفع المشروع على GitHub
إنشاء Web Service في Render
إضافة متغيرات البيئة
تحديد أمر التشغيل
uvicorn main:app --host 0.0.0.0 --port 10000

تفعيل Webhook
https://api.telegram.org/botTOKEN/setWebhook?url=APP_URL/webhook

طريقة الاستخدام
===========================

عرض النتائج
إدارة الحساب والمفضلة والنقاط

الدعم
==============================
